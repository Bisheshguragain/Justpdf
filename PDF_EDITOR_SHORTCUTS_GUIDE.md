## üéØ Common Workflows

### Add and Edit Text
1. Click **Text** tool
2. Click on canvas
3. Enter text (prompt appears)
4. **Double-click** to edit inline
5. Press **Enter** to save

### Move Multiple Items
1. Click first item
2. **Ctrl/Cmd+Click** other items
3. Drag any selected item
4. All move together

### Quick Duplication
1. Select annotation(s)
2. **Ctrl/Cmd+D**
3. Duplicates appear selected
4. Drag to position

### Copy Across Pages
1. Select annotation(s) on page 1
2. **Ctrl/Cmd+C**
3. Navigate to page 2
4. **Ctrl/Cmd+V**
5. Annotations appear on new page

## üé® Selection Modes

### Visual Indicators
- **Green border**: Single selection
- **Purple border**: Multi-selection
- **No border**: Not selected

### Selection Actions
- **Click**: Select one
- **Ctrl/Cmd+Click**: Add to selection
- **Click canvas**: Deselect all
- **Escape**: Deselect all
- **Ctrl/Cmd+A**: Select all on page

## üìã Clipboard Operations

### Copy (Ctrl/Cmd+C)
- Works with one or many annotations
- Copied to internal clipboard
- Paste button becomes enabled

### Paste (Ctrl/Cmd+V)
- Pastes with +20px offset
- Works on current page
- Pasted items auto-selected

### Duplicate (Ctrl/Cmd+D)
- Faster than copy/paste
- Creates immediate copy
- Duplicates auto-selected
- +20px offset applied

## üîß Advanced Tips

### Precision Work
- Use **Select** tool for moving
- **Double-click** for quick text edits
- **Ctrl/Cmd+D** faster than copy/paste
- **Ctrl/Cmd+A** to bulk select

### Multi-Page Editing
- Annotations are page-specific
- Copy on one page, paste on another
- Use navigation for multi-page edits
- Each page is independent

### Efficient Selection
- **Click + Ctrl/Cmd+Click** for multi-select
- **Drag** when multiple selected to move all
- **Delete** removes all selected at once
- **Escape** to quickly deselect

### Keyboard-First Workflow
```
1. Ctrl/Cmd+A (select all)
2. Ctrl/Cmd+C (copy)
3. Ctrl/Cmd+V (paste)
4. Ctrl/Cmd+D (duplicate)
5. Delete (remove unwanted)
6. Escape (deselect)
```

## üêõ Troubleshooting

### Can't edit text inline?
- **Try**: Double-click the text
- **Or**: Use edit button (‚úé)
- **Make sure**: Not in Text tool mode

### Multi-select not working?
- **Hold**: Ctrl (Windows/Linux) or Cmd (Mac)
- **While**: Clicking items
- **In**: Select mode only

### Copy/Paste not working?
- **First**: Select annotation(s)
- **Then**: Use keyboard shortcuts
- **Check**: Not focused on input field

### Buttons grayed out?
- **Copy/Duplicate**: Need selection first
- **Paste**: Need to copy first
- **Select**: Annotations to enable
