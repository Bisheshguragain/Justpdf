# PDF Editor - Advanced Features Guide

## Overview
The JustPDF PDF Editor now includes advanced editing capabilities with flexible text selection, multi-select, copy/paste, inline editing, and comprehensive keyboard shortcuts.

---

## üéØ Key Features

### 1. **Inline Text Editing**
- **Double-click** any text annotation to edit it directly on the canvas
- Edit text in place without using prompts
- Changes are saved when you click outside or press Enter
- Press **Escape** to cancel changes

**How to use:**
1. Double-click on any text annotation
2. Edit the text directly in the inline editor
3. Press **Enter** or click outside to save
4. Press **Escape** to cancel

### 2. **Multi-Select Annotations**
- Select multiple annotations by holding **Ctrl** (Windows/Linux) or **Cmd** (Mac) while clicking
- Move multiple annotations together
- Delete multiple annotations at once
- Copy and paste multiple annotations

**How to use:**
1. Click first annotation to select it
2. Hold **Ctrl/Cmd** and click additional annotations
3. All selected annotations can be moved, copied, or deleted together
4. Press **Ctrl/Cmd + A** to select all annotations on the current page

### 3. **Copy, Paste & Duplicate**
- **Copy** selected annotations to clipboard
- **Paste** annotations with a slight offset
- **Duplicate** creates immediate copies
- Works with single or multiple annotations

**Keyboard Shortcuts:**
- **Ctrl/Cmd + C**: Copy selected annotations
- **Ctrl/Cmd + V**: Paste annotations
- **Ctrl/Cmd + D**: Duplicate selected annotations

### 4. **Enhanced Selection System**
- Click to select single annotation
- Ctrl/Cmd + Click to toggle selection
- Selected annotations show green border
- Multi-selected annotations show purple border
- Resize handles appear on selected annotations

### 5. **Flexible Annotation Management**
- **Move**: Drag any annotation to reposition
- **Resize**: Use corner handles to resize
- **Edit**: Double-click text or use edit button
- **Delete**: Click delete button or press Delete/Backspace
- **Deselect**: Click on canvas or press Escape

## üêõ Troubleshooting

### Text not editable?
- Make sure you're **double-clicking** on the text
- Ensure text tool is not currently active
- Try clicking the edit button instead

### Can't select multiple annotations?
- Hold **Ctrl** (Windows/Linux) or **Cmd** (Mac) while clicking
- Make sure you're in **Select** mode

### Copy/Paste not working?
- First select annotation(s)
- Use keyboard shortcuts (Ctrl/Cmd + C/V)
- Make sure no input fields are focused

### Inline editor disappears?
- Clicking outside saves changes
- Press **Enter** to save intentionally
- Press **Escape** to cancel
