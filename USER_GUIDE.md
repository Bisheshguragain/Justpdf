## üìö Available Tools & Use Cases

### 1. **Compress PDF**
**When to use**: Your PDF is too large to email or upload
- Reduces file size while maintaining quality
- Works best on PDFs with images
- Note: Browser-based compression has limitations

### 2. **Merge PDF**
**When to use**: Combine multiple PDFs into one document
- Upload multiple files
- Drag to reorder before merging
- Perfect for combining reports, scans, or documents

### 3. **Split PDF**
**When to use**: Extract specific pages or divide a large PDF
- **Extract Pages**: Get pages 1-5 from a 100-page document
- **Single Pages**: Split each page into its own PDF
- **Equal Chunks**: Divide into equal parts (e.g., 3 files of 10 pages each)

### 4. **Rotate PDF**
**When to use**: Pages are sideways or upside down
- Rotate all pages or specific pages
- Choose 90¬∞, 180¬∞, or 270¬∞ rotation
- Fix scanned documents

### 5. **Delete Pages**
**When to use**: Remove unwanted pages from a PDF
- Specify pages to delete (e.g., "2,5,7-10")
- Clean up documents before sharing
- Remove blank or error pages

### 6. **Image to PDF**
**When to use**: Convert photos or scans to PDF format
- Upload JPG or PNG images
- Reorder images before conversion
- Choose page size (A4, Letter, or Original)

### 7. **Extract Pages** ‚ú® NEW!
**When to use**: Pull out specific pages from a PDF
- Extract by page numbers: "1,3,5-7,10"
- Extract page range: "5 to 20"
- Extract all odd or even pages
- Great for sharing specific sections

### 8. **Organize Pages** ‚ú® NEW!
**When to use**: Rearrange, rotate, or remove pages visually
- Drag and drop thumbnails to reorder
- Rotate individual pages
- Delete unwanted pages
- See visual preview before saving
- Perfect for reorganizing scanned documents

### 9. **Protect PDF** ‚ú® NEW!
**When to use**: Add password protection to sensitive documents
- Set a password to open the PDF
- Password strength indicator
- Files processed locally (password never sent online)
- **Note**: For highly sensitive documents, use desktop software

### 10. **Page Numbers** ‚ú® NEW!
**When to use**: Add page numbering to your PDF
- Multiple formats: "Page 1", "1", "Page 1 of 10", "1/10"

- Customize font size
- Perfect for reports, books, and presentations

## ‚ùì Troubleshooting

### "Error loading PDF"
- Ensure file is a valid PDF
- Try with a different PDF to rule out file corruption
- Check file isn't password-protected (use Unlock PDF first)

### Large files are slow
- Be patient - browser processing can be slower than desktop software
- Consider splitting large files before processing
- Close other browser tabs to free up memory

### Tool not working
- Ensure JavaScript is enabled
- Try refreshing the page
- Clear browser cache
- Try a different browser

### Download not starting
- Check browser's download settings
- Some browsers may block downloads - check for a notification bar
- Try again with a different browser
