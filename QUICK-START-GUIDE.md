### "Conversion failed" Error
- **Cause**: Scanned PDF (image-based)
- **Solution**: User needs to use OCR first
- **Message**: Clearly shown in error message

### Progress Bar Stuck
- **Cause**: Very large file or browser memory issue
- **Solution**: Refresh page, try smaller file
- **Prevention**: 20MB limit enforced

### Download Not Starting
- **Cause**: Browser blocking downloads
- **Solution**: Allow downloads in browser settings
- **Check**: Console for FileSaver.js errors

### Blank Word Document
- **Cause**: PDF has no extractable text
- **Solution**: Use text-based PDFs
- **Message**: FAQ explains limitation
